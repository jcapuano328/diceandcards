# diceandcards
React Native dice roll/card draw app

#### UI

##### Main
```
|-------------------------------------------|
|  Title                              | * i |
|-------------------------------------------|
|                                           |
|  <               dice                 >   |
|             < roll button >               |
|                                           |
|-------------------------------------------|
|                                           |
|      [deck] <shuffle button>  [card]      |
|                                           |
|-------------------------------------------|
```

Wouldn't it be interesting to have a dice tower? "touch" the tower and dice fall into the view...

##### Configuration

```
|-------------------------------------------|
|  < Title - Configuration                  |
|-------------------------------------------|
|                                           |
|      <      dice configuration   >        |
|                                           |
|-------------------------------------------|
|                                           |
|      <      card configuration   >        |
|                                           |
|-------------------------------------------|
```

```
|-------------------------------------------|

|                                           |
|-------------------------------------------|
```
